# 💸 MYNLI – Your AI Financial Assistant 🚀

Mynli is an AI-powered finance platform built with the latest web technologies to help users manage their spending, scan receipts, and gain real-time insights — all in one beautiful, secure, and blazing-fast interface.

---

## 🧠 Features

✅ **Advanced Analytics**  
Get detailed insights into your spending patterns powered by AI.

🧾 **Smart Receipt Scanner**  
Automatically extract data from receipts with OCR & GenAI integration.

📊 **Budget Planning**  
Create and manage intelligent budgets with personalized suggestions.

💼 **Multi-Account Support**  
Link all your accounts and cards to manage everything in one place.

💱 **Multi-Currency**  
Track finances across currencies with real-time conversions.

📈 **Automated Insights**  
Actionable financial insights and alerts, generated by AI.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Auth**: [Clerk.dev](https://clerk.dev)
- **Database**: [Prisma ORM](https://www.prisma.io) + PostgreSQL
- **Backend-as-a-Service**: [Supabase](https://supabase.io)
- **Background Jobs**: [Inngest](https://www.inngest.com)
- **Security**: [Arcjet](https://arcjet.com)
- **UI Library**: [shadcn/ui](https://ui.shadcn.dev)
- **Email**: [React Email](https://react.email), [Resend](https://resend.com)

---

## 📦 Installation

**1. Clone the repository:** https://github.com/meantaraa/ai-finance-management-app.git


**2. Install dependencies:** npm install


**3.Run the app:** npm run dev

---

## 🌐 Deployment

You can deploy using [Vercel](https://vercel.com):

1. Connect your GitHub repo
2. Set environment variables in the dashboard
3. Click **Deploy**

Live App: [https://mynli.vercel.app](mynli-ai-finance-manager.vercel.app)
